# Queue-less Shopping System

A web-based self-checkout system that eliminates supermarket checkout queues by allowing customers to scan products and manage their cart directly.

## 🚀 Features
- Barcode-based product scanning
- Real-time cart updates
- Firebase Realtime Database integration
- Dynamic price calculation
- Responsive user interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Realtime Database
- Firebase Hosting
